# _Game Programming with Usagi_

A book by Brett Chalupa.

Made with [mdBook](https://rust-lang.github.io/mdBook/index.html) v0.5.x.

## Writing

1. Install mdbook - `cargo install mdbook`
2. Start the server - `mdbook serve`

Preview the changes at http://localhost:3000

## Hosting

The site is hosted on Netlify and currently requires a manual deploy with
`just deploy`.
