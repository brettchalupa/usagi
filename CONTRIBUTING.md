# Contributing

Bug fixes and documentation improvements are welcome. Usagi intentionally has a
small feature set and simple API. If you want to add a feature to Usagi,
[open an issue to discuss it first](https://codeberg.org/brettchalupa/usagi/issues/new).

See DEVELOPING.md for more details on working with the underlying engine code.

Contributions made to Usagi are in the public domain.

This project does not accept contributions made by AI coding agents or LLMs.
